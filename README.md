# repo-for-100hires
My answers to the required questions in the hiring email as a junior growth marketing specialist at 100 Hires.
Tools I installed: Cursor and I installed two extentions in it which are Claude Code and Codex.
Steps I completed: Install Cursor app in my laptop, install two extensions mentioned above and log in to them, create Github account and connect it to my Cursor account, create repository in Github: slide add README on, edit and put the answers in, hit publish and push to GitHub, I then open the repo in Cursor.
Issues I ran into: All are pretty new to me. I didn't have Cursor app installed. I didn't know how to connect to those extensions and log in to each one of them, I didn't know how to open the repo in Cursor. I then went to watch tutorial videos, read several pieces of information, even asked chatGPT how to solve problems.
